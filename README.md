# Kore

Kore is a privacy-focused browser project with a Firefox-inspired multi-process architecture.

The v0.1.0 foundation starts with Rust components for HTML parsing and rustls-backed networking.

## Commands

```sh
./mach build
./mach test
./mach run
```

On Windows, run the same entry point through Python if executable scripts are not enabled:

```powershell
python .\mach build
```
